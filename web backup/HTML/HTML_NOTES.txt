CSS types 
=======
1.inline CSS : element specific  CSS
2.internal CSS  : page specific  CSS
3.external CSS : application specific CSS

=========  
     <h1 style=' '    >  </h1>

style preference 
===============
     Q) if iam applying inline css,internal css which will load first in browser ?
     A) inline 
    
     Q) if iam applying inline css,internal css and external CSS which will load first in browser ?
     A) inline 

     Q) if iam applying internal , external ? 
      A)   external 
    
     Q) if iam applying external internal ? 
      A) internal
